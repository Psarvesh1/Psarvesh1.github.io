JavaLearn

It is a java application which will help students to learn basic concepts of java. Students can study various topics
with their example provided by their teachers. Teachers will also provide Google’s & YouTube’s link related to that
topic so that students can learn more about that topic.

Students can post various things related to java in ‘feed’. On post student can like and comment on it. This project
also includes ‘editor’ where users can store and run java programs. Users can send queries to admin about the system. Teacher can create new assessment or tasks for students, which student can complete and submit before their last date
of submission.

Technologies:
  • - Java
  • - Swing
  • - MySQL
  • - HTML/CSS
